3. Ensuring Assessment Integrity: The Dual-Score System & Advanced Anti-Cheating Framework

To ensure the credibility and fairness of every assessment, our platform employs a sophisticated dual-score system and a robust anti-cheating framework. This approach is designed to be cheat-resistant, fair for honest candidates, and low-friction enough to not compromise the user experience.

Dual-Score Concept:

  • Skill Assessment Score (SAS): Based purely on a candidate's performance in assessments, weighted by recruiter-defined importance (technical, behavioral, etc.). This is the primary score for skill evaluation.
  • Fraud Mapping Score (FMS): A hidden score (0-100) visible only to recruiters that reflects the likelihood of cheating or fraudulent activity during assessments.

Dimensions of Fraud Mapping Score (FMS):

  • Behavioral Consistency: Analyzes whether the candidate's test behavior is human-like. Detected via AI-based activity monitoring (pauses, rapid switching, irregular clicks). Penalty Weight: Medium.
  • Tab/Window Switch Frequency: Tracks how often a candidate switches out of the test window. Detected via Browser behavior monitoring. Penalty Weight: High.
  • Copy/Paste Detection: Looks for pasted content during assessments. Detected via Clipboard activity capture, typing pattern mismatch. Penalty Weight: High.
  • Impersonation Detection: Verifies identity during video-based tasks or test-taking. Detected via Face ID match, typing biometric profile, liveness detection. Penalty Weight: High.
  • IP & Device Consistency: Detects test attempts from unfamiliar or multiple devices/locations. Detected via IP address logging, device fingerprinting. Penalty Weight: Medium.
  • Language Style Anomaly: If essay-type answers drastically differ from profile or practice logs. Detected via NLP anomaly detection, stylometry. Penalty Weight: Low-Medium.
  • Time-to-Answer Anomalies: Answers completed suspiciously fast or slow based on question complexity. Detected via Adaptive test engine tracks average speed per question. Penalty Weight: Medium.
  • Screen Recording/Remote Tool Use: Detects use of remote-control or screen-sharing tools (with user consent for monitoring). Detected via Background process monitoring. Penalty Weight: High.

Solutions to Detect & Prevent Cheating:

  • AI-based proctoring, browser lockdown, and real-time monitoring.
  • Typing biometrics, face ID, and device fingerprinting.
  • Adaptive test engines and NLP anomaly detection.
